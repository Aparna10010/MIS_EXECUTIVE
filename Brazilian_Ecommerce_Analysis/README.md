# 📊 E-Commerce Sales Performance Dashboard (Excel)

## 🔍 Project Overview
This project is an interactive E-commerce Sales Dashboard built in Microsoft Excel to analyze business performance using real-world transactional data. It covers **98K+ orders** and **R$1.68M revenue**, providing insights into sales trends, customer behavior, and payment patterns.

---

## 🎯 Objectives
- Analyze overall sales performance and revenue trends  
- Identify top-performing product categories  
- Understand customer purchasing behavior (AOV)  
- Evaluate payment method preferences  
- Analyze order distribution by day of the week  
- Enable interactive filtering for better decision-making  

---

## 📁 Dataset Description
The dataset was created by integrating multiple relational tables using SQL. Key columns include:

- Order ID  
- Customer ID  
- Order Date  
- Order Status  
- Product ID  
- Category Name  
- Price & Freight Value  
- Payment Type & Payment Value  
- Customer City & State  

---

## 🛠️ Tools & Techniques Used
- Microsoft Excel  
  - Pivot Tables & Pivot Charts  
  - Slicers & Timeline  
  - Data Cleaning & Transformation  

- SQL  
  - Data extraction, joins, and aggregation  

---

## 📊 Key KPIs
- **Total Orders:** 98,665  
- **Total Revenue:** R$ 1.68M  
- **Average Order Value (AOV):** R$ 170.41  
- **Top Payment Method:** Credit Card  

---

## 📈 Dashboard Features
- Monthly Revenue Trend  
- Top 10 Product Categories by Revenue  
- Revenue Distribution by Payment Method  
- Orders by Day of Week  
- Interactive Slicers (State, Category, Payment Type)  
- Timeline Filter for date-based analysis  

---

## 💡 Key Insights
- Credit card payments contribute the highest share of revenue  
- Revenue is distributed across a wide customer base (low dependency on top customers)  
- Orders are higher on weekdays compared to weekends  
- A few product categories contribute significantly to overall revenue
